# 06. 게시판

> 메뉴: **커뮤니티 → 게시판**

사용자가 작성한 게시글을 관리하고, 신고된 게시글을 처리합니다.

---

## 게시글 (커뮤니티 → 게시판 → 게시글)

### 볼 수 있는 정보
- 제목 / 작성자 / 댓글 수 / 좋아요 / 조회수 / 작성일 / 상태

### 탭 필터
- **전체** — 모든 게시글
- **공개** — 사용자에게 보이는 게시글만
- **숨김** — 어드민이 숨김 처리한 게시글
- **공지** — 공지로 고정된 게시글만

### 검색·필터
- 기간 (작성일 기준)
- 검색어 (제목 또는 작성자명)
- 25개씩 페이지네이션

### 할 수 있는 작업

| 버튼 | 동작 |
|------|------|
| **공지** | 게시글을 공지로 고정 → 사용자 목록 최상단에 노출 |
| **공지해제** | 공지 고정 해제 |
| **숨김** | 사용자 화면에서 안 보이게 처리 (해제 가능) |
| **숨김해제** | 다시 노출 |
| **삭제** | 게시글·댓글·좋아요 모두 영구 삭제 (복구 불가) |
| 제목 클릭 | 상세 페이지로 이동 |

### 상세 페이지
- 본문 / 첨부 이미지 / 댓글 목록 확인
- **댓글 개별 삭제** 가능
- 상단에서 공지 토글 / 숨김 토글 / 게시글 삭제 가능

> 💡 **공지 / 숨김 / 삭제 선택 기준**
> - 일시적인 부적절한 글: **숨김** 권장 (해제 가능)
> - 명백한 위반·스팸: **삭제** 권장
> - 운영 공지(점검 안내 등): **공지**로 고정

---

## 신고글 (커뮤니티 → 게시판 → 신고글)

사용자가 게시글을 신고하면 여기에 쌓입니다.

### 볼 수 있는 정보
- 신고 대상 게시글 제목 (클릭 → 게시글 상세로 이동)
- 게시글 작성자
- 신고자 닉네임 + UID
- 신고 사유
- 처리 상태 / 신고일

### 탭 필터
- **대기** — 처리 대기중 (이게 보통 가장 먼저 봐야 할 탭)
- **처리됨** — 숨김 또는 삭제로 처리한 신고
- **기각** — 사실무근으로 기각한 신고
- **전체**

### 처리 방법 (대기 상태)

| 버튼 | 동작 |
|------|------|
| **숨김** | 게시글을 숨김 처리하고 신고를 종결 |
| **삭제** | 게시글을 영구 삭제하고 신고를 종결 |
| **기각** | 게시글은 그대로 두고 신고만 기각 |

---

## 자주 묻는 질문

**Q. 사용자는 어떻게 신고하나요?**
- 게시글 상세 화면에서 **🚩 게시글 신고하기** 버튼을 클릭하고 사유를 입력합니다.
- 본인이 작성한 글은 신고할 수 없습니다.

**Q. 같은 게시글이 여러 번 신고됐어요.**
- 각 신고는 개별 항목으로 표시됩니다. 한 번 처리(숨김/삭제)하면 게시글 자체가 처리되지만, 다른 신고는 별도로 처리해야 합니다.

**Q. 신고를 잘못 기각했어요.**
- **기각** 탭에서 해당 신고를 다시 확인할 수 있습니다. 다시 처리하려면 개발자에게 문의하세요.

**Q. 숨김 처리한 게시글은 작성자에게 알림이 가나요?**
- 별도 알림은 가지 않습니다. 작성자가 자기 글이 안 보이는 걸 보고 인지하게 됩니다.
