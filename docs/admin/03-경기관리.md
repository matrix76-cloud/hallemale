# 03. 경기 관리

> 메뉴: **운영 → 경기 관리**

KBL/NBA 등 외부 경기 일정을 조회할 수 있습니다.

---

## 예정된 경기

> 경로: **경기 관리 → 예정된 경기**

앞으로 열릴 경기 일정을 확인합니다.

### 볼 수 있는 정보
- 리그 / 경기일시 / 홈팀 ↔ 어웨이팀 / 장소
- 자동 크롤링된 데이터 (수동 입력 불필요)

### 검색·필터
- 리그별 (KBL / NBA 등)
- 날짜 기간

---

## 지난 경기

> 경로: **경기 관리 → 지난 경기**

이미 끝난 경기 결과를 확인합니다.

### 볼 수 있는 정보
- 리그 / 경기일시 / 홈팀 점수 ↔ 어웨이팀 점수 / 결과
- 자동 크롤링된 데이터

---

## 동작 방식

- 경기 데이터는 **외부 사이트(네이버 스포츠 등)** 에서 자동으로 수집됩니다.
- 매일 새벽과 15분마다 자동 업데이트되므로 운영자가 따로 등록할 필요가 없습니다.

## 자주 묻는 질문

**Q. 경기 데이터가 안 보여요.**
- 자동 크롤링 일정에 따라 약간의 지연이 있을 수 있습니다.
- 1시간 이상 안 보이면 개발자에게 문의해 주세요.

**Q. 경기 결과가 잘못됐어요.**
- 외부 사이트의 데이터를 가져오므로, 원본이 잘못된 경우 자동으로 수정됩니다.
- 수동 수정이 필요하면 개발자에게 문의해 주세요.
