# 12. 약관·정책

> 메뉴: **시스템 → 설정 → 약관/정책**

마이페이지의 **개인정보 처리지침**과 **이용약관** 본문을 직접 작성·수정합니다.

---

## 사용 흐름

1. 메뉴 진입 → 자동으로 기본 템플릿이 화면에 채워져 있습니다.
2. 상단 탭에서 **개인정보 처리지침** 또는 **이용약관** 선택
3. 제목과 본문 수정
4. **저장** 버튼 → 사용자 화면에 즉시 반영

---

## 화면 구성

### 상단 탭
- **개인정보 처리지침** — 마이페이지 → 개인정보 처리지침에 노출
- **이용약관** — 마이페이지 → 이용약관에 노출

> 탭 전환 시 저장하지 않은 변경사항이 있으면 확인 메시지가 뜹니다.

### 입력 영역
- **제목** — 페이지 상단에 보일 제목 (기본값: 개인정보처리방침 / 서비스 이용약관)
- **본문** — 실제 약관/정책 내용

### 본문 작성 규칙
- **줄바꿈은 그대로** 사용자 화면에 보입니다.
- **빈 줄을 넣으면 문단**이 분리됩니다.
- 별도 마크다운/HTML 없이 텍스트로만 작성하셔도 됩니다.

### 메타 정보
- **마지막 수정 일시** — 가장 최근 저장 시각
- **수정자** — 어떤 어드민 계정이 수정했는지

### 버튼
| 버튼 | 동작 |
|------|------|
| **기본 템플릿** | 화면을 기본 본문으로 되돌림 (저장 안 함) |
| **되돌리기** | 마지막에 **저장**된 상태로 복귀 |
| **저장** | Firestore에 저장 + 사용자 화면 즉시 반영 |

---

## 사용자 화면 노출

저장 즉시 반영되며, 다음 위치에서 사용자가 볼 수 있습니다.

- 마이페이지 → **개인정보 처리지침** 메뉴
- 마이페이지 → **이용약관** 메뉴
- 회원가입 시 약관 동의 페이지

---

## 자동 시드

이 페이지에 처음 들어가면 기본 본문이 자동으로 등록되어 사용자 화면에 노출됩니다.
이후로는 어드민이 수정한 내용으로만 노출됩니다.

> ⚠️ **두 탭 모두 한 번씩 들어가야** 두 문서가 다 등록됩니다.
> (한 탭만 들어가면 다른 문서는 빈 상태가 됩니다)

---

## 자주 묻는 질문

**Q. 저장 안 한 상태로 다른 메뉴로 이동했어요.**
- 변경사항은 사라집니다. 다시 작성해 주세요.
- 다음 버전부터 페이지 이탈 시 경고를 추가할 수 있습니다.

**Q. 사용자가 변경된 약관에 다시 동의해야 하나요?**
- 현재는 별도 동의 절차 없이 바로 노출됩니다.
- 중대한 약관 변경 시 별도 공지로 안내하시는 것을 권장합니다.

**Q. 이전 버전 약관을 보존할 수 있나요?**
- 현재는 최신 버전만 저장됩니다.
- 이전 버전 보존 기능이 필요하시면 개발자에게 문의해 주세요.

**Q. 약관에 외부 링크나 이미지를 넣을 수 있나요?**
- 현재는 텍스트만 지원됩니다.
- HTML/마크다운 지원이 필요하시면 개발자에게 문의해 주세요.
