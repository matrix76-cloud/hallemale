# 08. 광고 관리

> 메뉴: **콘텐츠 → 광고 관리**

홈 화면에 노출되는 배너와 진입 시 뜨는 이벤트 팝업을 관리합니다.

---

## 홈 배너 (콘텐츠 → 광고 관리 → 홈 배너)

홈 상단에 슬라이드로 노출되는 히어로 배너입니다.

### 새 배너 등록

1. 우측 상단 **+ 새 배너 등록** 클릭 → 모달 오픈
2. 정보 입력
   - **이미지** (필수): 좌측 영역 클릭 → 이미지 업로드
     - 권장: **1200 × 600 px (가로:세로 = 2:1)** / 5MB 이하
     - 자동 1080px로 압축
     - 텍스트가 들어간 이미지보다 **여백이 있는 일러스트/사진** 추천
   - **제목** (필수): 줄바꿈은 `\n` 으로 입력 가능
     - 예) `팀 만들고\n오늘 한 판 어때?`
   - **설명**: 한 줄 설명
   - **이미지 위치**: 왼쪽 / 오른쪽 (텍스트와 좌우 배치)
   - **텍스트 정렬**: 왼쪽 / 오른쪽
   - **표시 순서**: 작을수록 슬라이드 앞쪽
   - **사용자에게 노출 (활성)** 체크박스
3. **배너 등록** 버튼 → 등록 완료

### 배너 수정·삭제·활성/비활성
- 목록에서 **수정** / **삭제** / **활성화/비활성화** 버튼 사용

### 활용 팁
- **3~5개** 배너를 두면 슬라이드가 자연스럽습니다.
- 시즌별 이벤트, 신규 기능 안내, 이용 가이드 등으로 활용하세요.

---

## 이벤트 팝업 (콘텐츠 → 광고 관리 → 이벤트 팝업)

홈 진입 시 화면 중앙에 뜨는 팝업입니다.

### 새 팝업 등록

1. **+ 새 이벤트 팝업 등록** 클릭 → 모달 오픈
2. 정보 입력
   - **이미지** (선택): 720 × 900 px (4:5) 권장 / 5MB 이하
     - 이미지 없이 텍스트만으로도 등록 가능
   - **제목** (필수): 예) `신규 가입 이벤트 진행 중!`
   - **본문** (선택): 팝업에 표시할 안내 텍스트
   - **버튼 라벨** (선택): 예) `자세히 보기`
   - **이동 URL** (선택): 외부 링크 또는 `/home/banners` 같은 내부 경로
   - **시작 일시** (선택): 비우면 항상 노출
   - **종료 일시** (선택): 비우면 항상 노출
   - **표시 순서**: 작을수록 우선
   - **사용자에게 노출 (활성)** 체크박스
3. **팝업 등록** 클릭 → 등록 완료

### 동작 방식
- 사용자가 홈에 진입할 때 활성 팝업이 표시됩니다.
- 사용자가 **"오늘 하루 보지 않기"** 를 누르면 그 날 동안 다시 뜨지 않습니다.
- 시작/종료 일시를 설정하면 해당 기간에만 노출됩니다.

### 팝업 수정·삭제·활성/비활성
- 목록에서 **수정** / **삭제** / **활성화/비활성화** 버튼 사용

---

## 자주 묻는 질문

**Q. 배너와 팝업의 차이는요?**
- **배너**: 홈 상단에 항상 노출되는 슬라이드. 가벼운 안내·홍보용.
- **팝업**: 홈 진입 시 화면 가운데에 뜨는 강조 알림. 중요한 이벤트나 공지용.

**Q. 동시에 여러 팝업이 활성화되면 어떻게 되나요?**
- 표시 순서가 작은 것부터 차례로 노출됩니다. 한 번 닫으면 다음 팝업이 뜹니다.

**Q. 비활성화하면 즉시 사용자에게서 사라지나요?**
- 네, 새로고침 시 즉시 반영됩니다.

**Q. 외부 URL이 아닌 앱 내부 페이지로 보내고 싶어요.**
- 이동 URL에 `/home/community` 처럼 슬래시(/) 로 시작하는 경로를 넣으면 됩니다.
