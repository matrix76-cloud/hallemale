# 05. 구장 관리

> 메뉴: **운영 → 구장 관리**

할래말래에서 추천하는 구장(체육관/실외 코트)을 등록·수정·삭제합니다.
사용자는 매칭 시 등록된 구장 중에서 선택할 수 있습니다.

---

## 새 구장 등록

1. 우측 상단 **+ 새 구장 등록** 버튼 클릭 → 모달 오픈
2. 정보 입력
   - **구장명** (필수): 예) 고려대 화정체육관
   - **주소** (필수): 예) 서울 성북구 고려대로 1
   - **상세 주소** (선택): 예) 지하 1층 농구코트
   - **위도 / 경도** (선택): 지도 표시용
   - **종류**: 실내(체육관) / 실외(코트) / 기타
   - **이용 비용**: 무료 / 유료 / 협의
   - **메모**: 이용 안내, 주차 정보 등
   - **표시 순서**: 작을수록 앞에 노출
   - **사용자에게 노출 (활성)** 체크박스
3. 좌측 이미지 영역 클릭 → 구장 이미지 업로드 (권장 1200×800)
4. **구장 등록** 버튼 클릭 → 등록 완료

---

## 구장 수정

- 목록에서 해당 구장의 **수정** 버튼 클릭 → 같은 모달이 열림
- 수정 후 **수정 저장** 버튼

---

## 구장 삭제

- 목록에서 **삭제** 버튼 클릭 → 확인 후 영구 삭제
- 삭제하면 사용자가 해당 구장을 더 이상 선택할 수 없습니다.

---

## 활성/비활성

- **사용자에게 노출 (활성)** 체크박스로 제어
- 비활성 상태인 구장은 사용자에게 노출되지 않지만, 어드민 목록에는 보입니다.
- 일시적으로 안 보이게 하고 싶을 때 활용

---

## 샘플 구장 자동 등록

운영 초기 테스트용으로 **샘플 2개 자동 등록** 버튼이 있습니다.
- 동서울대 체육관, 한국외대 미네르바콤플렉스 체육관
- 이미지·주소·좌표 포함
- **한 번만 실행**하세요. (중복 등록 방지)

---

## 자주 묻는 질문

**Q. 위도/경도는 어떻게 입력하나요?**
- 카카오맵이나 구글맵에서 구장 위치를 검색 → 위도/경도 복사
- 입력하지 않아도 등록은 가능하지만, 지도 표시에 제한이 있습니다.

**Q. 이미지를 안 올리면 어떻게 되나요?**
- 등록은 가능하지만, 사용자 화면에 기본 이미지가 표시됩니다.

**Q. 한 구장에 여러 사진을 올릴 수 있나요?**
- 현재는 한 장만 가능합니다.
