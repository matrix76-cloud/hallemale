# 00. 시작하기

> 처음 접속하시는 운영자를 위한 안내입니다.

---

## 1. 접속 방법

### 주소
- **운영 서버**: https://halle-bf789.web.app/admin/login

### 초기 로그인
| 항목 | 값 |
|------|-----|
| 아이디 | `admin` |
| 비밀번호 | `admin` |

> 🔐 **첫 로그인 후 반드시 비밀번호를 변경해 주세요.**
> 변경 방법: 좌측 메뉴 → **시스템 → 설정 → 운영자 계정** → **비밀번호 변경** 버튼

### 자동 로그인
로그인 화면에서 **자동로그인** 체크박스를 선택하면 다음 접속 시 로그인 없이 바로 들어갈 수 있습니다.

### 로그아웃
화면 우측 상단 **로그아웃** 버튼

---

## 2. 화면 구성

### 좌측 사이드바
모든 메뉴가 카테고리별로 정리되어 있습니다.

- **운영** — 회원, 팀, 경기, 매칭, 구장 관리
- **커뮤니티** — 게시판, 채팅 관리
- **콘텐츠** — 광고, 알림, 앱 업데이트
- **시스템** — 운영자 계정, 약관/정책

화살표(▼)가 있는 메뉴는 클릭하면 하위 메뉴가 펼쳐집니다.

### 상단바
- **좌측**: 현재 위치(빵부스러기 경로)와 접속 시각
- **우측**: 로그인된 운영자 이름 + 로그아웃 버튼

---

## 3. 대시보드

로그인하면 바로 보이는 화면입니다.

### 주요 통계
- **회원 수** — 전체 가입 회원 수
- **팀 수** — 등록된 팀 수
- **활성 매칭** — 진행 중인 매칭 수
- **오늘 가입자** — 오늘 새로 가입한 회원 수

> 숫자는 자동으로 갱신됩니다. 새로고침 시 최신값이 반영됩니다.

### 빠른 이동
대시보드의 카드를 클릭하면 해당 관리 페이지로 바로 이동할 수 있습니다.

---

## 4. 처음 운영자가 해야 할 일 (체크리스트)

배포 직후 한 번만 진행하시면 됩니다.

- [ ] **`admin` 계정 비밀번호 변경**
  - 시스템 → 운영자 계정 → admin 행의 **비밀번호 변경** 클릭
- [ ] **추가 운영자 등록 (필요 시)**
  - 시스템 → 운영자 계정 → **+ 운영자 추가**
- [ ] **약관·정책 검토**
  - 시스템 → 약관/정책 → 개인정보 처리지침 / 이용약관 본문 확인
  - 회사 정책에 맞게 수정 후 **저장**
- [ ] **홈 배너 1개 이상 등록**
  - 콘텐츠 → 홈 배너 → **+ 새 배너 등록**
- [ ] **추천 구장 등록**
  - 운영 → 구장 관리 → **+ 새 구장 등록**

---
