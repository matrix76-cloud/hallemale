# 01. 회원 관리

> 메뉴: **운영 → 회원 관리**

회원 목록 조회, 신고 처리, 차단 관리를 할 수 있습니다.

---

## 회원 목록

> 경로: **회원 관리 → 회원 목록**

### 볼 수 있는 정보
- 닉네임 / 이름 / 전화번호 / 지역 / 가입일
- 활동 통계 (참여 매칭 수 등)

### 검색·필터
- 검색어 (닉네임/이름/전화번호)
- 가입일 기간
- 25개씩 페이지네이션

### 할 수 있는 작업
- **상세 보기** — 행 클릭 시 회원의 상세 프로필 확인
- **차단** — 부적절 회원 즉시 차단

---

## 회원 신고

> 경로: **회원 관리 → 회원 신고**

다른 사용자가 회원을 신고하면 여기에 쌓입니다.

### 처리 방법

| 버튼 | 동작 |
|------|------|
| **차단** | 신고 사유를 검토 후 회원을 차단 (신고는 자동으로 종결) |
| **기각** | 신고 내용이 부적절하거나 사실무근일 때 (회원은 그대로) |

### 탭 필터
- **대기** — 아직 처리하지 않은 신고
- **차단됨** — 차단으로 처리된 신고
- **기각** — 기각된 신고
- **전체** — 모든 상태

### 차단 진행 시
- 차단 사유 입력 모달이 열림 → 신고 내용을 참고해서 사유 작성
- **차단하기** 버튼 클릭 → 회원이 차단되고 자동으로 차단 회원 페이지로 이동

---

## 차단 회원

> 경로: **회원 관리 → 차단 회원**

차단 처리된 회원을 모아서 볼 수 있습니다.

### 볼 수 있는 정보
- 닉네임 / 전화번호 / 지역
- 차단 사유 / 차단일 / 처리자

### 할 수 있는 작업
- **차단 해제** — 정당한 사유 확인 후 다시 활성화
- **상세 보기** — 차단된 회원의 프로필 확인

---

## 자주 묻는 질문

**Q. 차단된 회원은 앱을 사용할 수 있나요?**
- 차단된 회원은 로그인은 되지만 모든 기능이 제한됩니다. (자동 안내 화면 표시)

**Q. 신고를 잘못 차단했어요. 되돌릴 수 있나요?**
- **차단 회원** 페이지에서 해당 회원을 찾아 **차단 해제** 가능합니다.

**Q. 회원이 직접 탈퇴할 수 있나요?**
- 네. 마이페이지에서 회원 탈퇴가 가능합니다. 단, 탈퇴 후에는 복구되지 않습니다.
