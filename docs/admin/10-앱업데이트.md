# 10. 앱 업데이트

> 메뉴: **콘텐츠 → 앱 업데이트**

> 🚧 **이 기능은 현재 준비중입니다.**

---

## 향후 개발 예정 기능

### 새 버전 등록
- 버전 번호 (예: `1.0.3`) 입력
- 업데이트 내용 작성
- 등록 시 사용자 화면에 자동 토스트 알림 + 새로고침

### 버전 목록
- 등록된 모든 버전 이력 확인
- 수정·삭제

### 자동 새로고침 동작
- 사용자가 앱을 사용 중일 때 새 버전이 등록되면:
  1. 화면 하단에 "새로운 버전으로 업데이트 중..." 토스트 표시
  2. 1.5초 후 자동 새로고침
  3. 최신 코드로 갱신

### 마이페이지 표시
- 사용자 마이페이지의 **버전 정보** 항목에 최신 버전 번호 표시

---

## 임시 안내

현재는 코드 배포 시마다 사용자가 자동으로 최신 버전을 받게 됩니다.
(별도 등록 절차 없이 자동 적용)
