# 02. 팀 관리

> 메뉴: **운영 → 팀 관리**

팀 목록 조회, 신고 처리, 차단 관리를 할 수 있습니다.

---

## 팀 목록

> 경로: **팀 관리 → 팀 목록**

### 볼 수 있는 정보
- 팀 이름 / 팀장 / 지역 / 멤버 수 / 생성일
- 팀 활동 통계 (경기 수, 승률 등)

### 검색·필터
- 검색어 (팀명/팀장명)
- 지역 필터
- 25개씩 페이지네이션

### 할 수 있는 작업
- **상세 보기** — 행 클릭 시 팀 정보 미리보기
- **차단** — 부적절 팀 즉시 차단

---

## 팀 신고

> 경로: **팀 관리 → 팀 신고**

사용자가 팀을 신고하면 여기에 쌓입니다.

### 처리 방법

| 버튼 | 동작 |
|------|------|
| **차단** | 신고 사유 검토 후 팀 차단 |
| **기각** | 신고가 부적절하거나 사실무근일 때 |

### 탭 필터
- **대기** / **차단됨** / **기각** / **전체**

---

## 차단 팀

> 경로: **팀 관리 → 차단 팀**

차단된 팀을 모아서 봅니다.

### 할 수 있는 작업
- **차단 해제** — 다시 활성화
- **상세 보기** — 차단된 팀 정보 확인

---

## 자주 묻는 질문

**Q. 팀을 차단하면 팀 멤버들도 모두 차단되나요?**
- 아닙니다. 팀만 차단되며, 팀 멤버는 개인 계정으로 정상 활동 가능합니다.

**Q. 팀 차단 시 진행 중인 매칭은 어떻게 되나요?**
- 차단된 팀의 매칭은 자동으로 취소됩니다.

**Q. 팀이 해체되면 어떻게 되나요?**
- 팀장이 직접 해체하면 멤버는 자동으로 분리됩니다. 어드민 차단과는 별개입니다.
