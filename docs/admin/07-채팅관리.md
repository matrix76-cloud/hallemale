# 07. 채팅 관리

> 메뉴: **커뮤니티 → 채팅 관리**

사용자 간 1:1 채팅방을 모니터링하고 관리합니다.

---

## 채팅방 목록

### 볼 수 있는 정보
- 참여자 (닉네임 + UID 두 명)
- 마지막 메시지 / 마지막 메시지 시각
- 채팅방 생성일
- 상태(활성 / 잠금)

### 탭 필터
- **전체** / **활성** / **잠금**

### 검색·필터
- 기간 (생성일 기준)
- 검색어 (참여자 닉네임 / UID / 마지막 메시지 내용)
- 25개씩 페이지네이션

### 할 수 있는 작업

| 버튼 | 동작 |
|------|------|
| **보기** | 채팅방 상세로 이동 → 메시지 일람 |
| **잠금** | 사유 입력 후 잠금 → 사용자가 메시지를 보낼 수 없음 |
| **잠금해제** | 다시 사용 가능하게 풀어줌 |
| **삭제** | 채팅방 + 모든 메시지 영구 삭제 |

### 잠금 동작
- 잠긴 방에서 사용자가 메시지를 보내려고 하면 **"관리자가 잠근 채팅방입니다. 메시지를 보낼 수 없습니다."** 라는 에러가 뜹니다.
- 잠금 사유는 어드민 기록용이며 사용자에게 보이지 않습니다.

---

## 채팅방 상세

목록의 **보기** 또는 참여자 이름을 클릭하면 들어갑니다.

### 볼 수 있는 정보
- **상단**: 참여자 정보 (이름 + 아바타), 방 ID, 유형, 생성일, 잠금 상태/사유
- **메시지 일람**: 전체 메시지를 시간순으로 표시 (텍스트/이미지)

### 할 수 있는 작업
- **방 잠금 / 잠금 해제** (상단 우측 버튼)
- **방 삭제** (상단 우측 버튼)
- **메시지 개별 삭제** (각 메시지 우측 **삭제** 버튼)
- 메시지 이미지 클릭 → 새 탭에서 원본 보기

---

## 자주 묻는 질문

**Q. 채팅방에 참여해서 메시지를 보낼 수 있나요?**
- 어드민은 모니터링 전용이며, 메시지를 보낼 수는 없습니다.

**Q. 잠금된 방의 메시지도 볼 수 있나요?**
- 네, 잠금 상태와 무관하게 모든 메시지를 확인할 수 있습니다. 새 메시지만 차단됩니다.

**Q. 채팅방을 삭제하면 사용자에게 알림이 가나요?**
- 별도 알림은 가지 않습니다. 사용자가 채팅 목록에 들어갔을 때 방이 사라진 것을 보게 됩니다.

**Q. DM 외에 그룹 채팅도 있나요?**
- 현재는 1:1 DM만 있습니다.
