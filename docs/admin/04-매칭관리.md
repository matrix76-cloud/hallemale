# 04. 매칭 관리

> 메뉴: **운영 → 매칭 관리**

회원/팀 간 매칭 현황을 모니터링할 수 있습니다.

---

## 매칭 목록

> 경로: **매칭 관리 → 매칭 목록**

### 볼 수 있는 정보
- 매칭 ID / 호스트 팀 / 상대 팀 / 매칭일시 / 장소 / 상태

### 매칭 상태
- **모집중** — 호스트 팀이 상대를 찾고 있음
- **확정** — 매칭이 성사됨
- **진행중** — 경기 진행 중
- **종료** — 경기가 끝남
- **취소** — 매칭이 취소됨

### 검색·필터
- 상태 필터
- 매칭일 기간
- 25개씩 페이지네이션

### 할 수 있는 작업
- **상세 보기** — 매칭의 상세 정보 (라인업, 채팅, 결과 등)

---

## 분쟁/신고 (준비중)

> 경로: **매칭 관리 → 분쟁/신고**

> 🚧 **이 기능은 현재 준비중입니다.**
> 매칭 진행 중 발생한 분쟁이나 노쇼 신고를 처리할 예정입니다.

---

## 자주 묻는 질문

**Q. 매칭을 어드민이 강제 취소할 수 있나요?**
- 현재는 자동 모니터링 위주이며, 강제 취소 기능은 추가 개발 예정입니다.

**Q. 매칭 결과를 수정할 수 있나요?**
- 매칭 결과는 양 팀이 동의해야 확정됩니다. 운영자 임의 수정 기능은 향후 분쟁 처리 기능에 포함될 예정입니다.
