# 09. 알림·푸시

> 메뉴: **콘텐츠 → 알림/푸시**

공지사항을 작성하고 푸시 알림을 발송합니다.

---

## 공지 작성

> 경로: **알림/푸시 → 공지 작성**

사용자에게 보여줄 공지사항을 등록합니다.

### 새 공지 등록
1. **+ 새 공지** 또는 작성 버튼 클릭
2. 정보 입력
   - **제목** (필수)
   - **본문** (필수): 줄바꿈 보존
   - **카테고리** (선택): 일반 / 이벤트 / 점검 등
   - **표시 순서**
   - **활성 여부**
3. **등록** 버튼

### 공지 수정·삭제
- 목록에서 **수정** / **삭제** / **활성화/비활성화**

### 사용자에게 노출되는 곳
- 마이페이지 → **공지사항** 메뉴
- 알림 → **공지** 탭

---

## 푸시 발송 (준비중)

> 경로: **알림/푸시 → 푸시 발송**

> 🚧 **이 기능은 현재 준비중입니다.**
>
> 향후 개발 예정 사항:
> - 전체 회원 / 특정 회원 / 특정 팀 대상 푸시 발송
> - 즉시 발송 / 예약 발송
> - 발송 결과 자동 기록

### 임시 안내
- 현재는 **자동 푸시**만 동작합니다. (예: 채팅 메시지, 매칭 신청, 댓글 알림 등)
- 운영자가 수동으로 일괄 푸시를 보내려면 개발자에게 문의해 주세요.

---

## 발송 로그 (준비중)

> 경로: **알림/푸시 → 발송 로그**

> 🚧 **이 기능은 현재 준비중입니다.**
>
> 향후 개발 예정 사항:
> - 푸시·인앱 알림 발송 이력 조회
> - 발송 성공/실패 통계
> - 실패 사유 확인

---

## 자주 묻는 질문

**Q. 공지를 등록하면 푸시도 자동으로 가나요?**
- 현재는 별도입니다. 공지는 화면에만 등록되고, 푸시는 별도 발송이 필요합니다.
- 향후 "공지 등록 시 푸시 발송 옵션"을 추가할 예정입니다.

**Q. 사용자가 푸시를 안 받게 설정했어요.**
- 사용자가 마이페이지 → 알림 설정에서 직접 끌 수 있습니다.
- 끈 사용자에게는 발송되지 않습니다.

**Q. 푸시 발송 기능은 언제 추가되나요?**
- 추가 개발 일정은 개발자에게 문의해 주세요.
