# 11. 운영자 계정

> 메뉴: **시스템 → 설정 → 운영자 계정**

관리자 계정을 추가하고 비밀번호를 변경하거나 삭제할 수 있습니다.

---

## 운영자 목록

### 볼 수 있는 정보
- **아이디** (로그인 ID)
- **이름** (표시명)
- **권한** (최고 관리자 / 운영자)
- **등록일**

### 권한 종류
- **최고 관리자 (보라색)** — `admin` 계정. 시스템에서 자동 생성되며 **삭제할 수 없습니다**.
- **운영자 (초록색)** — 추가로 등록한 일반 운영자. 삭제 가능.

> 💡 본인 계정 옆에는 `(나)` 표시가 붙습니다.

---

## 운영자 추가

1. 우측 상단 **+ 운영자 추가** 클릭 → 모달 오픈
2. 정보 입력
   - **아이디**: 영문/숫자/`_`/`-` 3~20자 (예: `manager1`)
   - **이름**: 표시될 이름 (예: 홍길동)
   - **비밀번호**: 4자 이상
3. **추가** 버튼 → 즉시 해당 계정으로 어드민 로그인 가능

> 추가된 계정은 일반 **운영자** 권한입니다.

---

## 비밀번호 변경

1. 변경할 운영자 행의 **비밀번호 변경** 버튼 클릭
2. 새 비밀번호 입력 (4자 이상)
3. **변경** 버튼

> `admin` 본인 비밀번호도 동일한 방식으로 변경 가능합니다.

---

## 운영자 삭제

1. 삭제할 운영자 행의 **삭제** 버튼 클릭
2. 확인 → 즉시 삭제

### 삭제 불가 케이스
- **`admin` (최고 관리자)** — 시스템 보호로 삭제 불가
- **현재 로그인된 본인 계정** — 실수 방지로 삭제 불가
  (다른 어드민 계정으로 로그인한 다음 삭제 가능)

---

## 보안 권장사항

- 비밀번호는 **다른 서비스와 다르게** 설정하세요.
- **최소 8자 이상**, 영문 대소문자 + 숫자 조합 권장.
- 사용하지 않는 계정은 즉시 **삭제**하세요.
- 운영자 변경 시(퇴사 등) 즉시 비밀번호 변경 또는 계정 삭제.

---

## 자주 묻는 질문

**Q. 비밀번호를 잊어버렸어요.**
- 다른 어드민 계정으로 로그인 후 → 운영자 계정 페이지에서 비밀번호 변경 가능
- 모든 어드민 계정의 비밀번호를 잊은 경우 개발자에게 문의 (Firestore에서 직접 재설정 필요)

**Q. `admin` 계정의 이름을 바꿀 수 있나요?**
- 현재는 이름 변경 기능이 없습니다. 향후 추가될 수 있습니다.

**Q. 한 명에 여러 권한을 줄 수 있나요?**
- 현재는 **최고 관리자** / **운영자** 두 단계만 있습니다.
- 모든 운영자는 모든 메뉴에 접근할 수 있으며, 단지 `admin` 계정만 삭제 보호됩니다.

**Q. 운영자가 동시에 여러 명 로그인할 수 있나요?**
- 네, 각자 다른 PC에서 동시에 로그인 가능합니다.
