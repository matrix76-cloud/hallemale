## 1. 프로젝트 개요

### 1-1. 서비스 소개
**할래말래(HALLAEMALLAE)** 는 아마추어 **농구** 동호인을 위한 **팀 매칭 + 구장(체육관) 예약 + 커뮤니티/랭킹** 플랫폼이다.
"같이 농구 할래 말래?" — 팀과 팀, 선수와 선수를 연결해 **오늘 당장 뛸 수 있는 상대와 코트**를 찾아주는 것이 핵심이다.

React 기반 모바일 웹을 **React Native WebView 래퍼**(안드로이드/iOS 네이티브 앱, 저장소: `HallaMalle-mobile`)로 감싸 **모바일 웹 / Android / iOS 3개 플랫폼**을 동시 제공하며, PC 접속 시 별도 **랜딩페이지**(`web/index.html`)로 서비스를 소개한다. 백엔드는 **Firebase**(Auth·Firestore·Storage·FCM) + **Cloud Functions**를 사용한다.

## 3. 회원 및 권한 구조

### 3-1. 회원 권한 등급
| 등급 | 설명 | 인증 세션 |
|---|---|---|
| USER (일반회원) | 선수로 활동. 팀 생성/가입, 매칭, 예약, 커뮤니티 | 카카오/구글 소셜 + 전화번호 |
| 🟩【할래말래 추가】 팀 리더/멤버 | 팀 내부 역할. 리더는 멤버 초대·가입요청 승인·라인업 편성 권한 | USER 기반 |
| 🟩【할래말래 추가】 OWNER (구장주) | 구장 등록·예약 승인. **별도 이메일/비밀번호 로그인 세션**(소셜 아님) | `ownerAuth` |
| ADMIN (관리자) | 관리자 페이지 접근. Firebase `isAdmin` 클레임 기반 | 별도 `/admin/login` |

- 🟩【할래말래 추가】 **구장주 인증 분리**: 사용자앱 세션과 완전히 분리된 `ownerAuth` 세션. 미로그인 시 `/owner/login`으로 유도. 회원탈퇴 = 계정 완전 삭제.

### 3-2. 회원 가입 게이트 (순차 통과) 🟩【할래말래 추가: 게이트 체인】
로그인 후 다음 게이트를 **순서대로** 통과해야 본 서비스 진입:
1. **RequireAuth** — 로그인 여부
2. **RequireConsent** — 만 14세 이상 + 이용약관·개인정보처리방침 동의(`AgreementGate`)
3. **RequirePhone** — 전화번호 SMS 인증(`phoneVerified`). **전화번호를 키로 카카오/구글 동일인 계정 통합**
4. **RequireWelcome** — 가입 완료 화면 1회 노출(`welcomeSeen`)
5. **RequireClub** — 클럽/팀 컨텍스트 로딩

> 어드민 세션은 3·4단계 면제.

### 3-3. 회원 정책
- 신고 누적/약관 위반 시 관리자 차단(회원·팀·구장 각각).
- 차단 회원은 매칭·채팅·글쓰기 제한.

### 3-4. 탈퇴 후 데이터 처리
- 사용자 탈퇴: 개인정보 파기, 작성 콘텐츠는 정책에 따라 익명화/보관.
- 🟩【할래말래 추가】 구장주 탈퇴: **계정 완전 삭제**(`ownerWithdrawService`).

## 4. 🟩【할래말래 추가】 팀 · 선수 프로필 체계

> 시크원의 "업체(샵)" 개념을 할래말래에서는 **팀**과 **선수**로 이원화한다. 이 절 전체가 할래말래 고유.

### 4-1. 선수(개인) 프로필
- **포지션**: 가드(PG/SG) · 포워드(SF/PF) · 센터 — 포지션 칩으로 표시(홈 개인랭킹 칩 색상은 회색 `tone=text` 통일).
- **능력치**: `playerAbilityCalculator` 기반 스탯(득점·수비 등) 산출.
- **신체 정보**: 키·나이대 등(`MyProfileBodyEditPage`).
- **소개 / 미디어**: 자기소개, 하이라이트 사진·영상(`MyProfileIntroEditPage`, `MyProfileMediaEditPage`).
- **팀 가입 상태**: 소속 팀·초대·가입요청(`MyProfileTeamJoinEditPage`).
- **전적**: 개인 경기 기록(`PlayerMatchHistoryFullPage`).

### 4-2. 팀 프로필
- 팀 생성(`TeamCreatePage`) / 관리(`TeamManagePage`).
- 구성 섹션: 소개 · 멤버 · 라인업 · 미디어 · 하이라이트 · 전적 · 모집(리크루트).
- **멤버 초대**(`MyTeamInvites*`) / **가입 요청 승인**(`TeamJoinRequests*`).
- 팀 전적(`TeamMatchHistoryFullPage`).

### 4-3. 랭킹 (왕관 규칙)
- **팀 랭킹** / **개인(선수) 랭킹** 별도(`TeamRankingFullPage`, `PlayerRankingFullPage`).
- 🟩 **왕관 규칙(앱 전체 공통·동적)**: 랭킹 **1~3위** 팀/선수의 프로필 사진 위에 왕관 로고 배치. 홈 프리뷰·랭킹 전체 화면·프로필 어디서든 동일하게 적용.
- 홈 프리뷰: 팀/개인 랭킹 미리보기 섹션(`HomeRankingPreview`), 승리팀 티커(`WinningTickerBar`).
