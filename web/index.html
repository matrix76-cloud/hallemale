상대를 못 구함

팀은 있는데 붙을 상대가 없어서 결국 경기가 무산돼요.

일정·장소 혼선

누가 언제 어디서, 대화가 흩어져서 자꾸 헷갈려요.

실력 차이

막상 붙어보니 실력 차가 커서 경기가 재미없을 때가 많아요.

정산 번거로움

구장비 걷고 송금 받고, 매번 팀장이 챙기기 번거로워요.

기록이 안 남음

경기를 해도 전적·승률이 남지 않아 우리 팀 실력을 알 수 없어요.

핵심 기능

우리 팀의 경기를

01 추천 매칭

해볼 만한 상대를
골라줘요

우리 팀 실력과 지역을 분석해 붙어볼 만한 상대팀을 추천해요.

  • ✓지역·실력 기반 상대팀 추천
  • ✓"예상 승률 63%" 매칭 정보 제공
  • ✓전체 팀 리스트에서 직접 신청
02 매칭룸

매칭이 수락되면
매칭룸에서 확정

전용 매칭룸에서 날짜·장소·시간을 깔끔하게 정해요.

  • ✓매칭 성사 시 전용 채팅방 자동 개설
  • ✓일정·장소·시간 한 곳에서 확정
  • ✓누가 언제 어디서, 헷갈릴 일 없음
03 전적·기록

경기 결과가
전적이 돼요

경기마다 승·무·패가 팀 프로필에 자동으로 쌓여요.

  • ✓승률·연승·최근 전적 자동 기록
  • ✓상대 전적(H2H)까지 한눈에
  • ✓뛸수록 성장하는 우리 팀 데이터
04 지역 랭킹

우리 동네 1등,
누가 될까?

경기할 때마다 지역 랭킹이 실시간으로 바뀌어요.

  • ✓승리 +5 · 무승부 +2 · 패배도 +1점
  • ✓실시간 지역·전국 랭킹
  • ✓시즌 상위 팀엔 보상까지
05 팀장 일괄결제 준비 중

구장비, 팀장이
한 번에 결제

지금은 구장에서 현장 정산해요. 앱에서 바로 결제하는 기능을 준비하고 있어요.

  • ✓현재: 앱에서 예약 요청 → 구장 승인 → 현장 정산
  • ✓준비 중: 팀장 일괄결제로 송금 걷을 필요 없이 한 번에
  • ✓결제 기능 오픈 시점은 앱 공지로 안내드려요
구장

구장 잡는 것까지
앱 안에서 끝나요.

제휴 구장은 앱에서 바로 예약하고, 이미 쓰는 구장이 있으면 직접 입력해서 그대로 경기해요.

지도에서 근처 구장을
한눈에

지역과 일정으로 근처 제휴 구장을 찾고, 요금과 시설을 비교해요.

  • ✓지도·목록으로 근처 제휴 구장 탐색
  • ✓실내·실외, 코트 수, 시간당 요금 비교
  • ✓주차·샤워실 등 편의시설까지 확인

날짜와 시간을 고르면
예약 요청 끝

비어 있는 시간대를 눌러 예약을 요청하고, 구장이 승인하면 확정돼요.

  • ✓날짜별 빈 시간대를 눌러서 바로 선택
  • ✓연속된 시간대는 원하는 만큼 이어서 예약
  • ✓구장 승인 후 확정 · 이용료는 현장 정산

      FAQ

      자주 묻는 질문

      Q
      할래말래는 무료인가요?
      네, 매칭·전적·랭킹 등 앱 이용은 무료예요. 저희가 받는 이용료는 없습니다. 구장비는 이용 당일 구장에 직접 정산하시면 돼요. (앱에서 바로 결제하는 기능은 준비 중이에요.)
      Q
      팀이 아직 없어도 쓸 수 있나요?
      앱에서 팀을 만들거나 팀에 가입할 수 있어요. 팀을 꾸린 뒤 상대 매칭을 신청하면 됩니다.
      Q
      어떤 종목을 지원하나요?
      지금은 농구부터 시작해요. 반응을 보며 풋살·배구·테니스로 종목을 넓혀갈 예정입니다.
      Q
      경기 결과는 어떻게 반영되나요?
      경기가 끝나면 결과가 팀 전적과 지역 랭킹에 자동 반영돼요. 승률·연승·최근 전적이 프로필에 쌓입니다.
      종목

      지금은 농구부터.

      할래말래는 농구로 먼저 시작해요. 동네 농구인들의 반응을 보며, 풋살·배구·테니스로 종목을 하나씩 넓혀갈 거예요.
